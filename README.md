<div style="max-width: 800px; margin: 0 auto; line-height: 1.7;">

LibreCode is a high-performance Android harness that transforms mobile devices into autonomous engineering workstations. By bridging a native Android frontend with a robust Python/Flask backend via Chaquopy, it provides a desktop-class agentic experience without the overhead of Termux or secondary Linux layers.

<h2 style="color: #58a6ff; border-bottom: 1px solid #30363d; padding-bottom: 10px; margin-top: 40px;">Autonomous Vision and Web Analysis</h2>

The system features an integrated, agent-controlled browser specifically engineered for deep website analysis and manipulation. This is not a simple wrapper; it is a full interaction engine:

*   **Recursive Shadow DOM Traversal:** The agent can navigate and interact with elements buried deep inside complex web components (React, Lit, Vue).
*   **Heuristic Interaction Engine:** Identifies clickable areas and input fields using computed styles (cursor detection) and accessibility markers.
*   **Real-time Streaming Interception:** Full visibility into WebSocket and EventSource (SSE) traffic for debugging streaming applications.
*   **Precision Control Tools:** Native support for hover states, complex keyboard shortcuts, and multi-step form automation.

<h2 style="color: #58a6ff; border-bottom: 1px solid #30363d; padding-bottom: 10px; margin-top: 40px;">Premium Coding Models - Available for Free</h2>

LibreCode is built on the principle that high-tier intelligence should be accessible. The harness provides out-of-the-box access to the world's most capable coding and reasoning models at no cost:

*   **Qwen 3.6 Plus / Max:** State-of-the-art performance for complex logic and architectural design.
*   **MiniMax M2.5:** High-speed reasoning with exceptional reliability for code generation.
*   **GPT OSS 120B:** Massive-scale open source intelligence for broad technical knowledge.
*   **Gemini 3.0 Flash:** Ultra-low latency execution for rapid tool-use cycles.
*   **Llama 4 Scout:** Advanced instruction following for precise engineering tasks.
*   **DeepSeek V3.2:** Specialized technical reasoning for system-level programming.

<h2 style="color: #58a6ff; border-bottom: 1px solid #30363d; padding-bottom: 10px; margin-top: 40px;">Advanced Agentic Workflow</h2>

The agent operates through a rigorous "Plan then Build" cycle, ensuring every modification is deliberate and verified:

1.  **Context Indexing:** Uses native ripgrep and fd integration to map your local codebase.
2.  **Strategic Mapping:** Generates a detailed todo list of intended changes before writing a single line of code.
3.  **Mandatory Review:** Presents the execution plan for user approval to ensure alignment with project goals.
4.  **Surgical Patching:** Applies targeted edits instead of file rewrites, maintaining file history and minimizing side effects.
5.  **Autonomous Verification:** Leverages the integrated browser and shell tools to run tests and confirm implementation success.

<h2 style="color: #58a6ff; border-bottom: 1px solid #30363d; padding-bottom: 10px; margin-top: 40px;">Technical Stack</h2>

*   **Frontend:** Privileged Android WebView with bi-directional Java-to-JS bridge.
*   **Backend:** Local Flask server running Python 3.x via Chaquopy.
*   **Binary Tools:** Pre-compiled native Android binaries for ripgrep, fd, and toybox.
*   **Security:** Sandboxed local execution with granular permission management for storage and network.
