## 🛠️ Advanced Vision System

LibreCode's browser tool is engineered for complex modern web apps:
- **Recursive Shadow DOM:** Interacts with components in React, Lit, and Web Components that other agents miss.
- **Heuristic Detection:** Automatically identifies interactive `div`s and `span`s by analyzing computed styles and accessibility attributes.
- **Live Status Feedback:** Watch the agent's "thinking" in real-time via the blue status indicator in the browser header.

## 🏗️ Technical Stack

- **Frontend:** Responsive HTML/JS running in a privileged WebView.
- **Backend:** Python Flask server running via Chaquopy.
- **Tools:** Integrated `ripgrep`, `fd`, and native Android shell access.
- **Interaction:** Custom Java-to-JS bridge for precision element manipulation.

## 📜 How it Works

1. **Research:** The agent explores your project files and the web to gather context.
2. **Plan:** It generates a detailed todo list and awaits your green light.
3. **Execute:** Using surgical edits (not full rewrites), it implements features or fixes bugs.
4. **Verify:** It runs tests or uses the integrated browser to confirm everything works perfectly.
